FlashPunk was a popular AS3 game library for making Flash games (an alternative to the more popular Flixel library).

Included here is the last stable version (1.7.2) from Sep 19, 2013.

At least check other versions could be downloaded from https://github.com/useflashpunk/FlashPunk

At last check library documentation was still available at http://useflashpunk.net/
